📖 Panduan Admin – Sistem Pengaduan

KUA Kecamatan Panti  ·  Kabupaten Jember

🔐 Cara Login ke Panel Admin

1

Buka Tab "Panel Admin"

Klik menu Panel Admin di bagian atas halaman.

2

Masukkan Kredensial

Gunakan username dan password yang diberikan oleh pengelola sistem. Akun demo: admin / kua2025.

3

Klik Tombol "Masuk"

Setelah berhasil login, dashboard pengaduan akan ditampilkan secara otomatis.

💡 Catatan keamanan: Jangan bagikan username dan password kepada pihak yang tidak berwenang. Selalu klik Keluar setelah selesai menggunakan panel admin.

📋 Cara Melihat dan Mengelola Pengaduan

1

Lihat Ringkasan Statistik

Di bagian atas dashboard terdapat 4 kotak statistik: Total, Baru, Diproses, dan Selesai. Angka diperbarui otomatis.

2

Filter Pengaduan

Gunakan dropdown Status dan Jenis untuk menyaring data. Gunakan kolom pencarian untuk mencari berdasarkan nama atau nomor tiket.

3

Klik Tombol "Detail"

Klik tombol Detail pada baris pengaduan untuk melihat seluruh informasi: nama, NIK, WhatsApp, alamat, jenis, dan kronologi lengkap.

4

Update Status Pengaduan

Di dalam modal detail, ubah status menjadi Baru → Diproses → Selesai (atau Ditolak jika tidak memenuhi syarat), lalu klik Simpan Perubahan.

5

Export Data ke CSV

Klik tombol Export CSV untuk mengunduh seluruh data pengaduan dalam format spreadsheet (Excel/Google Sheets).

💡 Data pengaduan tersimpan sementara di memori browser (session). Untuk penyimpanan permanen, hubungi pengelola IT untuk integrasi database server.

🔄 Alur Penanganan Pengaduan

1

Terima Pengaduan (Status: Baru)

Pengaduan masuk otomatis berstatus Baru. Admin wajib mengecek dalam 1×24 jam.

2

Verifikasi & Tindak Lanjut (Status: Diproses)

Ubah status ke Diproses saat pengaduan sedang ditangani. Hubungi pelapor via WhatsApp untuk klarifikasi jika diperlukan.

3

Tutup Pengaduan (Status: Selesai / Ditolak)

Setelah ditangani, ubah ke Selesai. Jika pengaduan tidak memenuhi syarat atau terbukti tidak benar, ubah ke Ditolak disertai catatan alasan.

⏱ Target SLA: Pengaduan harus ditindaklanjuti paling lambat 3 hari kerja sejak diterima, sesuai standar pelayanan KUA Kecamatan Panti.
